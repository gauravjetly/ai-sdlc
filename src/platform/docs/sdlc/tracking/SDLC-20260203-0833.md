   - DevOps layer validation (pipelines, IAM)
   - AWS quota checking
   - IAM policy validation

   - Real-time cost estimation for 27 node types
   - Environment-specific pricing
   - Cost comparison across environments
   - Optimization suggestions
   - Budget tracking

3. **TerraformGenerator** (800 LOC)

   - Environment-specific variables
   - Backend configuration
   - Output generation

## Lessons Learned

1. The existing services (WorkflowManager, TemplateService, DesignService) were already well-designed and needed minimal enhancement
2. Terraform generation required careful handling of 27 different node types
3. Cost estimation required AWS pricing data (static for now, can be dynamic later)
4. Layer dependency resolution using topological sort ensures correct deployment order
