## Test Patterns

### Unit Test Pattern

```typescript
// Example: TemplateService unit test
describe('TemplateService', () => {
  let service: TemplateService;
  let prismaMock: MockPrismaClient;

  beforeEach(() => {
    prismaMock = mockDeep<PrismaClient>();
    service = new TemplateService(prismaMock);
  });

  describe('getTemplate', () => {
    it('should return template when found', async () => {
      // Arrange
      const mockTemplate = TemplateFactory.create();
      prismaMock.template.findUnique.mockResolvedValue(mockTemplate);

      // Act
      const result = await service.getTemplate(mockTemplate.id);

      // Assert
      expect(result).toEqual(mockTemplate);
      expect(prismaMock.template.findUnique).toHaveBeenCalledWith({
        where: { id: mockTemplate.id }
      });
    });

    it('should throw NotFoundError when template not found', async () => {
      // Arrange
      prismaMock.template.findUnique.mockResolvedValue(null);

      // Act & Assert
      await expect(service.getTemplate('non-existent'))
        .rejects.toThrow(NotFoundError);
    });
  });
});
```

## Architectural Decisions

### ADR-101: Test Framework Selection

**Decision**: Use Jest for unit/integration tests, Playwright for E2E tests.

**Rationale**:
- Jest: Industry standard, excellent TypeScript support, built-in mocking
- Playwright: Cross-browser support, reliable, modern API

**Alternatives Considered**:
- Vitest: Faster but less mature ecosystem
- Cypress: Good for E2E but heavier weight than Playwright

### ADR-102: Test Data Strategy

**Decision**: Use factory pattern with Faker.js for test data generation.

**Rationale**:
- Consistent, reproducible test data
- Easy to customize for specific scenarios
- Avoids hardcoded test data

### ADR-103: Mock Strategy

**Decision**: Mock external dependencies, test against real database for integration tests.

**Rationale**:
- Unit tests: Full mocking for speed and isolation
- Integration tests: Real database for accuracy
- E2E tests: Full stack for user journey validation
