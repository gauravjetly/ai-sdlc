### 3.2 Deploy Phase

```bash
# Deploy to staging
aws s3 sync dist/ s3://catalyst-staging-frontend/ --delete

# Invalidate CloudFront cache

  --distribution-id E1XXXXXXXXXXXX \
  --paths "/*"
```
