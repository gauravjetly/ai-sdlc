#### AC-004: Architecture Validation

| Given | When | Then | Status |
|-------|------|------|--------|
| Domain layer forbids infrastructure imports | Domain file imports from infrastructure | HIGH violation with line number | PASS |
| Domain layer forbids express | Domain file imports express | HIGH violation with remediation | PASS |
| Valid layer dependencies | Application imports from domain | Validation passes | PASS |

**Test Execution**:
```typescript
// Bad code: src/domain/BadService.ts
import { db } from '../infrastructure/database';

// Result:
// [HIGH] architecture.layer_violation
//   domain layer cannot import from infrastructure layer
//   at src/domain/BadService.ts:1
//   Fix: Move the dependency to an allowed layer or use dependency injection.
```
