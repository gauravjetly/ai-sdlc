## Bug Fixes Applied During Testing

### BUG-001: LRU Cache TTL=0 Expiration
- **Symptom**: `cache.set('key', 'value', 0)` followed by `cache.get('key')` returned the value instead of null
- **Root Cause**: Expiration check used `>` instead of `>=`. With TTL=0, `expiresAt = Date.now() + 0`, and the check happens in the same millisecond
- **Fix**: Changed `Date.now() > entry.expiresAt` to `Date.now() >= entry.expiresAt` in `get()`, `has()`, and `cleanup()` methods
- **Verification**: Test passes consistently

### BUG-002: SQLite HAVING Clause on Non-Aggregate Query
- **Symptom**: `searchKnowledge()` threw `SqliteError: HAVING clause on a non-aggregate query`
- **Root Cause**: Used `HAVING relevance_score > 0` without GROUP BY. SQLite strictly enforces this
- **Fix**: Moved keyword match conditions into WHERE clause using OR-joined LIKE expressions. Score remains in SELECT for ORDER BY
- **Verification**: All 17 memory provider tests pass
