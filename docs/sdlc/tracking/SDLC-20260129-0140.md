## Request

Build a **reusable, component-based DevOps automation platform on AWS** that covers the complete product lifecycle from provisioning to continuous operations.

### Core Principles
- **Reusable Building Blocks**: Linkable, Mappable, Composable, Parameterized, Testable, Versioned
- **Complete Lifecycle Coverage**: Provisioning, Deployment, Supportability, Continuous Operations

### Lifecycle Phases to Cover

1. **Provisioning Phase**
   - Infrastructure provisioning (VPC, compute, storage, databases)
   - Tenant provisioning (multi-tenant isolation)
   - Environment provisioning (Dev, Test, UAT, Prod)
   - Service provisioning (new microservices, APIs)

2. **Application Deployment Phase**
   - Code deployment (automated CI/CD)
   - Database migrations
   - Configuration management
   - Feature flag management

3. **Supportability Phase**
   - Monitoring and alerting
   - Centralized logging
   - Distributed tracing
   - Health checks and diagnostics
   - Backup and recovery automation

4. **Continuous Operations Phase**
   - Auto-scaling
   - Self-healing
   - Continuous deployment
   - Release management
   - Cost optimization
   - Security patching

### Technology Stack
- **Infrastructure**: Terraform + CloudFormation + AWS CDK
- **Pipeline**: AWS CodePipeline, GitHub Actions
- **Containers**: EKS (Kubernetes) + ECS
- **Serverless**: Lambda + Step Functions
- **Monitoring**: CloudWatch + X-Ray
- **Configuration**: Parameter Store + Secrets Manager
