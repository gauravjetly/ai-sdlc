## Deployment Steps

The `install-platform.sh` script executes 8 steps:

1. **Check prerequisites**: Node.js 20+, npm, Claude Code (optional)
2. **Create directory structure**: `~/.claude/` and `~/.aisdlc/`
3. **Install 12+ agent definitions**: Copy .md files to `~/.claude/agents/`
4. **Install slash commands**: Copy .md files to `~/.claude/commands/`
5. **Install platform packages**: npm install for storage and CLI packages
6. **Initialize SQLite database**: Apply migration schema
7. **Write platform configuration**: Generate `config.json`
8. **Install support scripts**: Registry CLI, FinOps tracker, version file

## Rollback Procedure

Since this is a local installation, rollback is straightforward:

```bash
# Full rollback
bash uninstall-platform.sh --confirm
git checkout <previous-version-tag>
bash install-platform.sh

# Data-preserving rollback
bash uninstall-platform.sh --keep-data
git checkout <previous-version-tag>
bash install-platform.sh
```
