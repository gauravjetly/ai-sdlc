## Usage

### As a Library
```typescript
import {
  HybridClassifier,
  SmartRouter,
  GovernanceEngine,
  DEFAULT_CONFIG,
} from '@aisdlc/integration';

// Classify a request
const classifier = new HybridClassifier({ tier2Enabled: false });
const classification = await classifier.classify('Add user authentication');

// Route to agents
const router = new SmartRouter();
const decision = router.route(classification, 2);

// Check governance
const engine = new GovernanceEngine({ level: 2 });
const govDecision = engine.evaluate(classification);
```

### As a CLI Tool
```bash
cd src/integration
npx ts-node cli/interactive-demo.ts
```
